# Clasificación de Prendas con Machine Learning

## 📌 Descripción del proyecto

Este proyecto implementa un sistema de clasificación de imágenes de prendas de vestir utilizando Machine Learning.

Las categorías son:
- Pantalón
- Playera
- Sudadera

---

## 🧠 Modelos utilizados

- Regresión Logística
- Random Forest

---

## ⚙️ Proceso

1. Carga y preprocesamiento de imágenes
2. Conversión a vectores
3. Entrenamiento de modelos
4. Optimización con GridSearchCV
5. Validación cruzada (K-Fold)
6. Registro de experimentos con MLflow

---

## 📊 Métricas utilizadas

- Accuracy
- Precision
- Recall
- F1-score

---

## 🧪 Resultados

Se compararon ambos modelos, obteniendo mejor desempeño el modelo de Random Forest.
